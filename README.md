# Recettes Mobile

Application mobile de recettes développée avec React Native et Expo.

L’application utilise l’API REST TheMealDB pour afficher les catégories de recettes, rechercher des plats et consulter les détails d’une recette.

## Fonctionnalités

- Affichage des catégories de recettes
- Affichage des plats d’une catégorie
- Recherche d’une recette par nom
- Navigation entre plusieurs écrans
- Affichage des détails d’une recette
- Affichage des ingrédients et des quantités
- Affichage des instructions de préparation
- Accès à la vidéo YouTube de la recette
- Gestion du chargement
- Gestion des erreurs avec bouton Réessayer
- Interface adaptée aux zones sécurisées du téléphone

## Technologies utilisées

- React Native
- Expo SDK 54
- React Navigation
- JavaScript
- TheMealDB API
- Expo Go
